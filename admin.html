⚡ API 配置说明

智能小结功能支持以下 API 提供商（按优先级自动切换）：
1. 七牛云 AI - 环境变量: QINIU_AI_API_KEY
2. Google Gemini - 环境变量: SUMMARY_GEMINI_API_KEY
3. 心流平台 (iFlow) - 环境变量: IFLOW_AI_API_KEY 免费API

系统会按上述顺序尝试调用，如当前API失败会自动切换到下一个。心流平台是免费备选方案。
